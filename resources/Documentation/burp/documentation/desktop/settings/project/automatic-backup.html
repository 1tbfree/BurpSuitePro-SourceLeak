Automatic backup settings

The Automatic backup settings enable you to configure automatic project file backups for disk-based projects. To enable backups, select Automatically back up the project every [x] minutes and specify the required time interval.

Note

You can only configure automatic backups for disk-based projects. If you are using a temporary project then the option to enable automatic backups is grayed out.

You can also control some aspects of Burp's behavior when running automatic backups:

  • Whether to include in-scope items only.
  • Whether to show a progress dialog during backups.
  • Whether to delete the backup file when Burp is shut down.

The Automatic backup settings are user settings. They apply to all installations of Burp on your machine.
