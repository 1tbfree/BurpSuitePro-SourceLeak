Logging settings

The Logging settings enable you to configure which of Burp's tools can add HTTP requests and responses to the log files. You can select to log requests, responses, or both from the following tools:

  • Proxy.
  • Scanner.
  • Repeater.
  • Intruder.
  • Sequencer.
  • Extender.

You can also select All tools to log from all of the tools in the list.

The Logging settings are project settings. They apply to the current project only.
