Burp Target tool

The Target tool enables you to define which targets are in scope for your current work. In the Site map, you can toggle between the URL view and the Crawl paths view. These views show you detailed information about your target applications. You can use the information about your target application's content and functionality to drive the workflow for your penetration testing.
