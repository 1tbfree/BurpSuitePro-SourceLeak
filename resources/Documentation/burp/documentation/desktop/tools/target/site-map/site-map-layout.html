Editing the site map layout

You can customize the layout of the URL view to make it easier for you to analyze the information.

Contents and issues layout

You can display the contents and issues in separate tabs, or you can split the pane vertically. To change the layout:

  1. Right-click the Contents or Issues pane.
  2. Select View and choose from Left/right split or Tabs.

Adding a new site map window

You can open a new site map in a new window, based on the same underlying data:

  1. Right-click the Contents or Issues pane.
  2. Select Show new site map window.

You can use the new window to display and monitor a different selection of target items. You can also apply a different display filter.
