Testing input validation

Input validation is performed by websites to ensure that data entered into the website is valid. If input validation is inadequate, an attacker may be able to use HTTP requests to inject malicious data into the vulnerable website.

Burp Suite includes a range of tools that enable you to test for input validation vulnerabilities.
