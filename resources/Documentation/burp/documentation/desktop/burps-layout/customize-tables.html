Customizing Burp's tables

You can customize most of the tables in Burp. This enables you to focus on the information that you're interested in. Burp remembers your preferences, so you don't need to reorganize your tables every time you start Burp.

Where applicable, you can make the following changes:

  • Hide columns - Right-click the header of the column you want to hide, then select Hide column.

  • Show hidden columns - Click the options menu > Table layout, then select the columns you want to display.

  • Move columns - Drag and drop the header of the column you want to move to its new location.

  • Sort the table - Click the header of the column you want to sort by. Each click cycles the sorting order through ascending, descending, and unsorted.

  • Filter the data - Click the Filter settings bar, set your criteria using the predefined checkboxes and fields.

  • Restore the default layout - To return the table to its original state, click the options menu > Table layout, then select Restore default table.

Advanced table customization

Some Burp tools have Bambda functionality, enabling you to customize tables in more advanced ways. This includes:

  • Advanced filtering - Click the Filter settings bar, then go to Bambda mode to create your own powerful custom filters.

  • Adding custom columns - Click the options menu > Add custom column to create a personalized column that displays the data you want to see.
