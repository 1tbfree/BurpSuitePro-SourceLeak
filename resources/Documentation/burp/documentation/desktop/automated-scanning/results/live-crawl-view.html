Live crawl view

When you select a scan from the Dashboard, the Live crawl view tab provides a rendered view of the page that Burp Scanner is currently crawling, as it would normally appear in a browser. This can help you gain a better understanding of how the crawler interacts with your target site. It is also useful for troubleshooting unexpected behavior during the crawl phase so that you can fine-tune your configuration.
