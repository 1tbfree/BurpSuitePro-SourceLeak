Reporting scan results

After you use Burp Scanner to scan a target, you can generate a report in HTML format for some or all of the issues found. This enables you to share the results with colleagues or clients. You can also export issue data in XML format, so that you can incorporate it into your own reporting systems.

You can use the reporting wizard to generate and customize the output.
