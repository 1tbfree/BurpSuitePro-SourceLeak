UI settings

The User interface settings enable you to configure aspects of Burp Suite's appearance. From here, you can change settings for:

  • Side panel - Configure the layout of the side panel and the Inspector widgets.
  • Message editor - Configure the layout of the message editor and how messages are displayed.
  • Hotkeys - Configure hotkey shortcuts for common actions.
  • Displays - Configure Burp Suite's theme and font size, and the tabs displayed on the menu.
