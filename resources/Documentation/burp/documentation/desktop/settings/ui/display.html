Display settings

The Display settings enable you to configure aspects of Burp Suite's appearance.

Appearance

The Appearance settings enable you to:

  • Select either a light or dark theme.
  • Select the font size for Burp's menus.

Note

These settings do not affect the font used to display HTTP messages. You can specify the HTTP message font in the HTTP message display settings.

The Display settings are user settings. They apply to all installations of Burp on your machine.

Learn tab display

This setting controls whether the Learn tab is displayed on the main navigation bar. This tab contains links to useful tutorials and other resources to help you get started with Burp Suite and discover new features.

The Learn tab display setting is a user setting. It applies to all installations of Burp on your machine.

Scaling

These settings help you to optimize Burp's appearance on high-resolution displays that use custom scaling. You do not need to alter these settings unless you are experiencing issues with Burp's appearance on a high-resolution monitor.

To specify manual scaling settings, select Configure scaling settings.

There are two options available:

  • Use system DPI settings means that Burp attempts to automatically scale based on your current display and system scaling settings.
  • Scale factor enables you to manually specify a scale for Burp to run at. Select the required scale from the list.

To revert to default scaling settings, click the refresh icon.

All scaling changes take effect the next time you restart Burp.
