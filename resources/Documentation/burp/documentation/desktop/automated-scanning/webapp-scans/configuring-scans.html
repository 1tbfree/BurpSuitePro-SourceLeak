Configuring web application scans

This section explains how to customize web application scans to meet your specific testing requirements. You can customize the following aspects of your scan:

  • The scan scope and configuration.

  • Authentication for your web application. You can provide sets of user credentials or specify full login sequences so that Burp Scanner can discover and audit content that is only accessible to authenticated users.

  • The resource pool in which the scan will be run.
