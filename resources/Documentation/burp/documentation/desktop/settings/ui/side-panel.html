Side panel settings

The Side panel settings page is located under User interface in the Settings menu.

The Side panel settings enable you to edit the layout of the default side panel and the Inspector widgets.

The Side panel settings are user settings. They apply to all installations of Burp on your machine.

Default side panel layout

These settings enable you to set the default layout for the side panel.

You can set the panel's default position as Left or Right.

You can set the panel's display mode:

  • Auto-expand - By default, the side panel expands and collapses with the available screen space.
  • Always collapsed - Enable this setting to keep the side panel collapsed.

Inspector widgets

These settings enable you to adjust how widgets are displayed in the Inspector. For each widget, you can:

  • Show - Enable this setting to show the widget in the Inspector
  • Open by default - Enable this setting to expand the widget in the Inspector by default
  • Wrap text - Enable this setting to wrap the text inside the widget cells

To reorder the list of widgets, use the Up and Down buttons.
