Reviewing unrequested items

You can use the site map to review unrequested items. These are items that the live passive crawl detected, but you have not yet requested.

The site map shows unrequested items in gray. To quickly locate unrequested items, select the whole application in the tree view and click the header of the Time requested column. This groups the unrequested items together.

You can manually review these items (for example, by copying each URL into Burp's browser) to see if they contain interesting content.
